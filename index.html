How It Works

1

User Agent Detection

The server checks the User-Agent header for curl, wget, or CLI indicators

2

Content Negotiation

For CLI clients, returns text/plain with ANSI escape codes instead of HTML

3

Beautiful Output

Terminal receives formatted text with colors, boxes, and emojis
